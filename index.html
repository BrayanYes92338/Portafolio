Conoce a Brayan Yesid Caballero

Soy Brayan Yesid Caballero, Analista y Desarrollador de Software Full Stack. Mi enfoque está en el análisis de requisitos y el desarrollo de soluciones tecnológicas que combinan eficiencia y creatividad. Tengo experiencia en la creación de aplicaciones web y sistemas backend utilizando tecnologías como Node.js, Vue.js y bases de datos NoSQL. Me apasiona resolver problemas complejos y aplicar prácticas de desarrollo ágiles para crear productos innovadores y funcionales.
